## Philosophy & Target Audience

Official examples demonstrating WebGPU API usage:
- Canonical API usage patterns
- Progressive complexity
- Covers most WebGPU features

Target audience: Developers learning WebGPU.

## Key Samples to Study

- **Basic rendering** — Hello Triangle, rotating cube
- **Textures** — Texture sampling, cubemaps
- **Compute** — Compute shaders, particle systems
- **Advanced** — Deferred rendering, shadows
