# orx (openrndr extensions)

> Extensions and utilities for openrndr.

## Quick Facts

| Property | Value |
|----------|-------|
| **Language** | Kotlin |
| **License** | BSD-2-Clause |
| **Repository** | [openrndr/orx](https://github.com/openrndr/orx) |

## Philosophy & Target Audience

orx extends openrndr with specialized functionality:
- Modular extensions
- Easy to include/exclude
- Community contributions welcome

## Key Extensions

- **orx-noise** — Noise functions
- **orx-shapes** — Shape utilities
- **orx-color** — Color manipulation
- **orx-panel** — UI panels
- **orx-fx** — Post-processing effects
- **orx-mesh-generators** — Mesh creation
