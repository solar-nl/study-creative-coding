# OPENRNDR Ecosystem

Libraries and extensions for the OPENRNDR creative coding framework.

## Libraries

| Library | Description |
|---------|-------------|
| [orx](./orx/) | Official extension library collection for OPENRNDR |

## About OPENRNDR

OPENRNDR is a Kotlin-based creative coding framework with a modern DSL-style API. The orx extensions provide additional functionality including noise generation, GUI panels, video processing, and more.
