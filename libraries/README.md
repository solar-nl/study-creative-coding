# Libraries

Reusable libraries organized by ecosystem.

## Ecosystems

| Ecosystem | Libraries | Description |
|-----------|-----------|-------------|
| [openrndr-ecosystem/](./openrndr-ecosystem/) | orx | Kotlin extensions for OPENRNDR |
| [processing-ecosystem/](./processing-ecosystem/) | toxiclibs | Java libraries for Processing |
| [web/](./web/) | threejs, pixijs, babylonjs | JavaScript rendering libraries |
| [rust/](./rust/) | wgpu | Rust GPU abstractions |
| [universal/](./universal/) | mixbox | Cross-platform utilities |

## Quick Reference

| Library | Language | Focus |
|---------|----------|-------|
| [orx](./openrndr-ecosystem/orx/) | Kotlin | Extensions for OPENRNDR (noise, GUI, compositor) |
| [toxiclibs](./processing-ecosystem/toxiclibs/) | Java | Geometry, physics, color theory |
| [threejs](./web/threejs/) | JavaScript | WebGL/WebGPU 3D graphics |
| [pixijs](./web/pixijs/) | JavaScript | WebGL 2D renderer |
| [babylonjs](./web/babylonjs/) | JavaScript | WebGL/WebGPU 3D engine |
| [wgpu](./rust/wgpu/) | Rust | WebGPU implementation |
| [mixbox](./universal/mixbox/) | Multi | Pigment-based color mixing |
