# Universal Libraries

Cross-platform libraries available in multiple languages.

## Libraries

| Library | Languages | Description |
|---------|-----------|-------------|
| [mixbox](./mixbox/) | C++, JS, Rust, GLSL, Python | Pigment-based color mixing |

## About Universal Libraries

These libraries aren't tied to any specific framework or ecosystem. They provide focused functionality that can be integrated into any project regardless of language or platform.

## Mixbox

Mixbox implements Kubelka-Munk pigment mixing theory, allowing digital colors to blend like real paint. Available as:

- C++ header-only library
- JavaScript module
- Rust crate
- GLSL shader include
- Python package
